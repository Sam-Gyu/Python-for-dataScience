# Python Data Science Basics

This repository contains my initial projects and learning notebooks for Data Science. It covers the fundamental building blocks of the Python data ecosystem.

## Topics Covered

* **Python Fundamentals**: Core syntax and data structures used in data analysis.
* **NumPy**: Efficient numerical computing and multi-dimensional array manipulation.
* **Pandas**: Data manipulation, cleaning, and analysis using DataFrames.

## Tech Stack
* **Language**: Python 
* **Libraries**: NumPy, Pandas
* **Environment**: Google Colab / Jupyter Notebooks

## Learning Goals
The goal of these notebooks is to build a strong foundation in data handling.
